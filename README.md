# Quadro Kanban - Projeto Web I

## Descrição
Este projeto consiste no desenvolvimento de um quadro Kanban interativo utilizando HTML, CSS e JavaScript.

O sistema permite criar tarefas, movimentá-las entre colunas com Drag and Drop e editar o texto dos cards com duplo clique.

## Funcionalidades
- Criar novas tarefas
- Organizar tarefas em três colunas:
  - A Fazer
  - Fazendo
  - Concluído
- Arrastar tarefas entre colunas
- Editar tarefas com duplo clique
- Layout responsivo para computador e celular

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Estrutura do Projeto
- `index.html` → estrutura da aplicação
- `style.css` → estilização e responsividade
- `script.js` → lógica da aplicação

## Conceitos aplicados
- Manipulação do DOM
- Eventos JavaScript
- API Drag and Drop do HTML5
- Responsividade com CSS Grid, Flexbox e Media Query
- Organização modular do código

## Como executar
1. Baixe ou clone o repositório
2. Abra a pasta no VS Code
3. Execute com a extensão Live Server
